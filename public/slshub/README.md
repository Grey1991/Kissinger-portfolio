## Image Guidelines

- **Format:** PNG preferred (JPG acceptable)
- **Naming:** Use exact filenames as specified above (lowercase, hyphens)
- **Size:** Optimize for web (max 2MB per image)
- **Aspect Ratio:** 16:9 recommended for consistency
- **Quality:** High enough to read text clearly when zoomed

## Temporary Placeholders

If you don't have all screenshots yet, you can:
1. Use placeholder images with the correct naming
2. Duplicate similar screenshots temporarily
3. Leave sections commented out until images are ready

The modal will handle missing images gracefully by showing broken image placeholders.
